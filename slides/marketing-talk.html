What is Camofox?

API browsing for agents
with anti-detection.

// open a page
POST localhost:9377/tabs
{ "url": "https://instagram.com/some_venue" }

// get page content
GET localhost:9377/tabs/{id}/snapshot
→ { "snapshot": "heading 'Some Venue' ... 1,234 followers ..." }
Facebook and Instagram block bots aggressively.
Camofox looks like a real browser — your agents browse without getting banned.

07 / Learnings

What I learned.

LinkedIn cannot be automated directly — use external APIs.
Claude + Camofox is too slow for content creation.
It re-discovers the UI every time — which button to click, where to type. 2 minutes + 2,000 tokens per post.
Solution: build dedicated scripts. Hard-coded clicks, no LLM needed. 5 seconds, zero tokens.
